This Program in C will get as a arguments an open socket(File Descriptor)
and start a P2P chat with this socket

notice.
you will need to open this program both on "sender" and reciver" each of them will have a terminal that communicates with user.

This program accepts via args(argv[1]) an FD of TCP Socket and Make a Synchronic “conversation”
between the two hosts that share the SOCKET, in any given time they can enter “exit” keyword and the
conversation will be over and the socket will be closed.

enjoy.
